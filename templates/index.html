          FAQ

          Frequently asked questions

          AbstractAI supports both structured abstracts (IMRAD format with explicit section labels like Background, Methods, Results) and unstructured abstracts (continuous prose). The AI automatically detects the format and adapts its analysis accordingly.
          Scores are generated by IBM Granite based on a detailed evaluation prompt that instructs the model to assess each dimension (Structure, Clarity, Completeness, Technical Depth, Novelty, Publication Readiness) on a 0–100 scale, following academic publishing standards.
          No. Your abstract is sent only to the local LangFlow server running on your machine (localhost:7860). Nothing is stored in a database or transmitted to external third-party services. Each session uses a unique UUID to prevent cross-session data mixing.
          The tool requires at least 50 characters to proceed. In practice, meaningful evaluation requires a full abstract — typically 150–300 words. Very short inputs may yield generic feedback.
          Install LangFlow (pip install langflow), run langflow run, then import your IBM Granite flow. Ensure the flow URL and API key in app.py match your configuration. Start this Flask app with python app.py.
          Yes. Click the PDF button in the evaluation dashboard header to download a formatted report of the full evaluation, including scores, components, strengths, weaknesses, and recommendations.
